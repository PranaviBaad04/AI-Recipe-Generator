# 🍽️ AI Recipe Generator

An AI-powered Recipe Generator built with **React.js** that creates delicious recipes based on the ingredients entered by the user. The application uses an AI API to generate recipes instantly and maintains a history of previously generated recipes during the session.

---

## 📌 Features

- 🥕 Enter available ingredients.
- 🤖 Generate recipes using AI.
- 📜 Recipe history displayed in the application.
- 🎨 Modern and responsive user interface.

---

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- Vite
- CSS / Inline Styling

### Backend/API
- Node.js
- Express.js
- Groq API
